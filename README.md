# RAG PDF Chatbot

A simple Retrieval-Augmented Generation (RAG) chatbot that allows users to ask questions about a PDF document.

## Features

- PDF text extraction using PyMuPDF
- Text chunking
- Embedding generation with Sentence Transformers
- Similarity search using FAISS
- Interactive Gradio interface

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```
